Portfolio

These introductary projects helped to lay the basis for creating console applications and use RAW SQL to interact with a database
  • Math Game - a simple console app to present maths questions to the user and validate responses
  • Calculator - allow the user to enter numbers and an operation and present the result, whilst learning multiple projects in a solution
  • Habit Logger - track a habit (drinking water) in an sqlite database
  • Coding Tracker - implement an external library (ConsoleTableExt) to display database in console

Moving Forward these projects extend the knowledge database usage in a console app and introduce APIs
  • Flashcards - using DTOs and SQL server present the user with a flashcard learning program, allowing the manipulation of the flash cards in the stack
  • Drinks - consume an online API via HTTP requests to show the user a drinks menu including ingredients
  • Phone Book - introduces Entity Framework CRUD operations in the form of a telephone book
  • Shifts Logger - create a Web API and consume it in a console app to log shifts worked

Push the basic .NET knowledge further
  • Exercise Tracker - Introduced to the Repository Pattern design pattern implementing a tracker for exercises
  • Sports Results - create a web crawler to scrape a static page for baseball results, compile and send daily the results by mail
  • Excel Reader - read data from a fixed Excel sheet into an SQL database for presentation in the console
  • Unit Tests - prepare a set of unit tests in the MSTest framework for the earlier Coding Tracker project to validate the functions

Using the knowledge gained in the console projects time to add some front-ends
  • Fitness Tracker - scaffold up a website using Razor Pages to allow users to log their fitness activites
  • Movies - using ASP.NET MVC with entity framework, build a front end to present movie information
  • Todo List - uses a minimal API and javascript fetch calls for a single page application
  • Budget - Create a relational database to track personal financies and their categories

Projects using the Angular front end framework

  • Coffee Tracker - an Angular web app with a backend webapi to log coffee consumption
  • Sleep Tracker - an Angular Material frontend including a sleep timer to record sleep sessions, presented in a paginated table
  • Quiz Game - an Angular Material quiz app where users can create their own quizzes and play them

Projects using the React front end framework
